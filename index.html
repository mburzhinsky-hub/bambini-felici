Ozon · надёжный режим импорта

Аналитика магазина по одному нормализованному файлу

Вместо загрузки сырых выгрузок Ozon приложение читает один подготовленный Excel-шаблон. Это убирает проблемы со сложными шапками, merged cells и нестабильными xlsx-структурами.

Месяц нужен для подписи отчёта и проверки, что книга собрана корректно.

Нужен файл с вкладками: Начисления, Заказы, Продвижение, Возвраты, Реализация.

Как теперь работает импорт

Один шаблон вместо пяти сырых отчётов

  1. Открой исходные выгрузки Ozon в Excel.
  2. Скопируй только данные в соответствующие вкладки шаблона.
  3. Не меняй заголовки на строке 1 и не вставляй служебные блоки.
  4. Сохрани один файл и загрузи его сюда.
Если xlsx в браузере недоступен, ниже есть резервный режим — загрузка пяти CSV из этого же шаблона.
